Project Description:

This project required us to build a circuit using a sensor in a voltage divide to change the state of an output device. My schematic explains the values chosen. Other requirements included using analogRead(),analogWrite(), the map function, and an if-statement in our code.

When it detects that the light levels are low, it fades the LED on (or off depending on if it detects light). It's pretty snazzy I must say! But how did we get here? First we have to gather our hardware. This project was built using the Arduino, breadboad, one red LED, an ohm resistor, a photoresistor and jump wires. Then I did some some calculations to measure appropriate resistor voltage and map equations. The calculations for those can be found below:

Photoresistor

V = I x R 5V = 5mA. x R R = 1k ohms ; value of resistor used in project

Red LED

Red LED = 2.4V ; forward voltage 5V - 2.4V = 2.6V ; resistor voltage Desired current = I = 15mA Required resistance = R = V/I = 2.6/0.015 = around 167 ohms. I used a 330 resistor for this project.

                        int ledPin = 10;
                        int pResistor = A0;
                        int sensorValue = 0;
                        int sensorMAX, sensorMIN;
                        
                        void setup() {
                          pinMode(ledPin, OUTPUT);
                          while (millis() < 3000) {
                            sensorValue = analogRead(pResistor);
                            if (sensorValue > sensorMAX) {
                              sensorMAX = sensorValue;
                            }
                            if (sensorValue < sensorMIN) {
                              sensorMIN = sensorValue;
                            }
                            Serial.begin(9600);
                          }
                        }
                        
                        void loop() {
                          sensorValue = analogRead(pResistor);
                          Serial.println(sensorValue);
                        
                          if (sensorValue > 60 && sensorValue < 150) {
                            digitalWrite(ledPin, LOW);
                          } else {
                            analogWrite(10, map(sensorValue, 250, 60, 5, 150));
                          }
                          delay(5);
                        }
